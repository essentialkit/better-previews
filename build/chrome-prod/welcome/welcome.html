Better Previews

Perform search and preview links in sidebars instead of opening new tabs.

Preview "hello world"

Usage:

1. For Search Previews

  • Select the text you would like to search for on the page.

  • A tooltip with the text "Search" would become visible, click the tooltip.

  • Alternatively, right click on the selected text and click "Preview Search Results".

  • Try it on this text "how tall is everest?"


2. For Link Previews.

  • Hover over the link you would like to preview.

  • A tooltip with the text "Preview" would become visible, click the tooltip.

  • Alternatively, right click on the selected text and click "Preview Link".

  • Try it on this link https://wikipedia.org

That's it!
